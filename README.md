# Geology-Constrained Machine Learning Framework for TOC Prediction

## Overview

This repository contains the core Python scripts used in the study:

**“A Modular Geology-Constrained Machine Learning Framework for TOC Prediction in Source Rocks: Cross-Basin Application and Robustness Assessment”**

The repository provides basin-specific training workflows for TOC prediction across three case-study datasets:

- **CN_train_energy_models.py** — training workflow for the Energies multi-basin dataset
- **egyptTrain.py** — training workflow for the Egypt Western Desert dataset
- **BakkenTrain.py** — training workflow for the US Bakken dataset

The main purpose of this repository is to provide the computational workflow and code structure used in the study.
